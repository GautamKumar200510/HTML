# 🌐 HTML Learning Repository

A beginner-friendly repository containing HTML basics, examples, and mini projects. This repository is part of my web development learning journey and serves as a quick reference for HTML concepts.

## 📚 Topics Covered

- HTML Document Structure
- Headings & Paragraphs
- Links & Images
- Text Formatting Tags
- Comments
- Horizontal Rule
- Subscript & Superscript
- Basic HTML Examples
- Mini Portfolio Project

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/GautamKumar200510/HTML.git
```

Open any `.html` file in your browser or VS Code to explore the examples.
